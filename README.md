Sweet Treats Store

Overview:

Sweet Treats Store is a full stack web application built using Node.js, Express.js, MongoDB Atlas, and Mongoose. The application demonstrates CRUD functionality for an e-commerce product database and includes a simple frontend interface for viewing products.

Features:

GET, POST, PUT, and DELETE product endpoints
MongoDB Atlas database integration
Product data stored with storeId, storeName, productId, productName, and price
Frontend product display page
Automated testing using Node.js
Deployment using Render
Running the Application

Install dependencies:

npm install

Start the server:

node server/server.js

Open a browser and navigate to:

http://localhost:3000
